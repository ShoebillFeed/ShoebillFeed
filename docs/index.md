# Shoebill Feed

Shoebill Feed is a self-hosted, LLM-powered news aggregator. It pulls articles from
RSS/Atom feeds, Reddit, YouTube, email newsletters, Mastodon, Bluesky, Lemmy,
GitHub, Telegram, arXiv, and arbitrary scraped web pages into a single feed,
uses a local or cloud LLM to summarize and categorize everything, and learns
which topics you actually care about from your own feedback — no ads,
no tracking, and nothing sold to advertisers.

This site documents how to run it, how it works internally, and how to
extend it.
